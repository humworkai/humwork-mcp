# Lovable Setup

Lovable uses a UI-based configuration — no config file needed.

## Steps

1. Open **Settings** in Lovable
2. Go to **Connectors → Personal connectors**
3. Click **New MCP server**
4. Enter:
   - **Name:** `humwork`
   - **Server URL:** `https://api.humwork.ai/api/v1`
5. Click **Save**

## Authentication

After adding the server, Lovable will prompt you to authenticate. Sign in with your Humwork account.

## Notes

- Personal connectors are only used during app creation (not deployed with the app)
- Enterprise plans may require an admin to enable MCP connectors
